# 📘 Miniguia de Estudos: Data Analytics com NotebookLM

Este caderno temático foi desenvolvido com foco no **aprendizado de Data Analytics**, utilizando ferramentas de **Inteligência Artificial** para apoiar o processo de estudo e prática.  
O tema foi escolhido por sua relevância no mercado de trabalho atual e pela necessidade crescente de profissionais capazes de interpretar dados e gerar insights estratégicos.

---

## 🎯 Objetivos de Estudo

1. **Compreender os fundamentos de Data Analytics**  
   - Estatística básica  
   - Coleta e organização de dados  

2. **Explorar ferramentas de IA aplicadas ao estudo**  
   - Uso do [NotebookLM](https://notebooklm.google.com/) para estruturar conteúdos  

3. **Desenvolver habilidades práticas**  
   - Manipulação de datasets  
   - Criação de dashboards  
   - Interpretação de métricas  

4. **Aplicar o conhecimento em casos reais**  
   - Simulações de análises de negócios  
   - Tomada de decisão baseada em dados  

5. **Construir um guia de referência pessoal**  
   - Reunir materiais, exemplos e anotações  

---

## 🛠️ Ferramentas Utilizadas
- NotebookLM  
- Python (Pandas, NumPy, Matplotlib)  
- Power BI / Tableau
